# hfit-campus-network-auto-login
# 🎓 HFIT 校园网自动登录脚本

> 合肥理工学院校园网自动登录油猴脚本，全自动填充账号密码、选择运营商，可视化提示运行状态。

## ✨ 功能特性
- ✅ **模拟真实键盘输入**：绕过校园网防自动填充机制
- ✅ **自动选择运营商**：支持中国电信/移动/联通切换
- ✅ **可视化状态提示**：页面右上角绿色/红色提示，无需打开控制台
- ✅ **超时保护机制**：10秒未找到输入框自动停止脚本，避免无效轮询
- ✅ **无弹窗干扰**：全程无alert弹窗，仅控制台输出日志
- ✅ **版权声明完整**：明确作者与使用许可，保护原创权益

## 🚀 快速开始

### 1. 安装油猴插件
- Chrome/Edge：安装 [Tampermonkey](https://www.tampermonkey.net/)
- Firefox：安装 [Greasemonkey](https://www.greasespot.net/)

### 2. 安装脚本
1. 复制本仓库中 `hfit-campus-network-auto-login.uesr.js` 完整代码
2. 打开油猴插件 → 「添加新脚本」
3. 粘贴代码 → 「文件」→「保存」

### 3. 配置个人信息
在脚本顶部修改以下信息：
```javascript
// ========== 仅需修改这里 ==========
const YOUR_STUDENT_ID = "你的学号";
const YOUR_PASSWORD = "你的校园网密码";
const TARGET_OPERATOR = "运营商"; // 可改为中国移动/中国联通/中国电信
const AUTO_CLICK_LOGIN = true;
const SCRIPT_TIMEOUT = 10000; // 超时时间（毫秒）
// ================================
```

### 4. 使用
打开校园网登录页 `http://10.10.2.15/`，脚本会自动执行：
- 自动填充学号、密码
- 自动选择运营商
- 自动触发登录
- 页面右上角显示运行状态提示
